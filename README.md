# Developer Portfolio - Emanuel Zimmermann

This repository contains the source code for my personal developer portfolio, which is deployed via GitHub Pages.

## Project Overview
The portfolio is designed to showcase my projects, skills, core technologies, and professional certifications. It serves as a central hub for recruiters to learn more about my background, current training path, and technical expertise.

## Built With
* **HTML5** - Semantic markup
* **CSS3** - Custom styling, media queries, and responsive design
* **Bootstrap 5.3.3** - Grid system, utility classes, and layout components

## Features
* **Fully Responsive Layout** - Optimized for mobile, tablet, and desktop devices
* **Bilingual Support** - Seamless switching between English and German versions
* **Interactive Elements** - Dynamic skill cloud, mapped location link, and smooth anchor scrolling
* **Legal Compliance** - Integrated legal notices (Impressum) for both language versions
